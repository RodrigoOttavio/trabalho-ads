# Sobre o Projeto
### Tema: Vida na água
<p align="left">Este é um projeto para proporcionar acessibilidade aos pescadores de pequena escala a locais 
reservados para pesca e fornecer sugestões ao mercado de pesca.</p>

# Status do Projeto 
 <p>Em desenvolvimento :warning:</p>
